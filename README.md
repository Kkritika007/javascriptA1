# JavaScript Assignment 1 – Interactive Web Application

This project is a simple interactive web application developed using **HTML, CSS, and JavaScript**. The application demonstrates the use of JavaScript to handle user interactions and dynamically update content on a webpage.

## 📖 Overview

The goal of this assignment is to practice core **JavaScript concepts** such as event handling, DOM manipulation, and interactive user interfaces.
The webpage allows users to interact with elements on the page, and JavaScript processes these interactions to display results or update the interface.

## 🚀 Features

* Interactive user interface
* Dynamic content updates using JavaScript
* Event-based user interaction
* Simple and responsive webpage design

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript**

## 💻 How to Run

1. Clone the repository

```
git clone https://github.com/Kkritika007/javascriptA1.git
```

2. Open the project folder.

3. Open **index.html** in any web browser.

## 🧑‍💻 How It Works

1. The webpage loads the HTML structure and styles from CSS.
2. JavaScript handles user interactions such as button clicks or input events.
3. Based on the user's actions, the page dynamically updates content.
